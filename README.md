# MapReduce: Вычисление среднего и дисперсии цен на недвижимость

Проект реализует вычисление математического ожидания (среднего) и выборочной дисперсии цен на недвижимость из датасета **New York City Airbnb Open Data** в парадигме MapReduce.

## О проекте

В рамках проекта реализованы четыре программы на C++:
- **mapper_average** — извлекает цены из 10-го столбца CSV файла
- **reducer_average** — вычисляет среднее арифметическое
- **mapper_variance** — вычисляет квадраты отклонений от среднего
- **reducer_variance** — вычисляет дисперсию

Архитектура следует классической парадигме MapReduce:
1. **Map** — извлечение данных и предварительные вычисления
2. **Shuffle** — автоматическая передача данных через pipe
3. **Reduce** — агрегация и финальные вычисления

## Структура проекта
```
├── bin/ # Скомпилированные исполняемые файлы
├── input/ # Директория для входных данных
│ └── AB_NYC_2019.csv # Датасет Airbnb NYC
├── mapper_average.cpp # Map для вычисления среднего
├── reducer_average.cpp # Reduce для вычисления среднего
├── mapper_variance.cpp # Map для вычисления дисперсии
├── reducer_variance.cpp # Reduce для вычисления дисперсии
├── run_locally_average.sh # Скрипт запуска для среднего
├── run_locally_variance.sh # Скрипт запуска для дисперсии
├── Makefile # Сборка проекта
└── README.md # Документация
```

## Требования

- Компилятор с поддержкой C++11 (g++, clang)
- Make
- Bash (для скриптов запуска)

## Установка и запуск

### 1. Подготовка данных

Поместите файл датасета ```AB_NYC_2019.csv``` в директорию ```input/```

### 2. Сборка проекта

Скомпилируйте все программы:
```
make clean
make all
```

После сборки в папке ```bin/``` появятся четыре исполняемых файла:
- ```mapper_average```
- ```reducer_average```
- ```mapper_variance```
- ```reducer_variance```

### 3. Вычисление среднего (математического ожидания)

Запустите скрипт для вычисления среднего:
```
./run_locally_average.sh
```

#### Результат:
- Значение среднего выводится в консоль
- Сохраняется в файл ```output_average``` в корне проекта

### 4. Вычисление выборочной дисперсии

После получения среднего, запустите вычисление дисперсии:

```
./run_locally_variance.sh
```
#### Результат:
- Значение выборочной дисперсии выводится в консоль
- Сохраняется в файл ```output_variance``` в корне проекта
